
SYSTEM INSTALLATION

1. Clone the repository and save it in your C:/xampp/htdocs folder.

2. Once cloned, open up command prompt and change directory to the project folder.

3. Then type "composer update". If there is an error, just re-run the command again.

4. Next type "npm install". If there is an error, just re-run the command again.

5. Now open up XAMPP and start APACHE and MYSQL services.

6. Next open up your web browser and go to "localhost/wcmc/public".

7. DONE!
